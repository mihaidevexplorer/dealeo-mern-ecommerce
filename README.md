# üõçÔ∏è Dealeo ‚Äì Full-Stack E-commerce Platform

**Dealeo** is a complete e-commerce platform built with **React**, **Vite**, **Redux**, **Express.js**, and **MongoDB**. It consists of three separate applications:

- ‚úÖ **Frontend** ‚Äì Customer-facing online shop  
- ‚úÖ **Backend (API)** ‚Äì Express server managing data and logic  
- ‚úÖ **Dashboard (Admin Panel)** ‚Äì Admin interface for managing products, orders, users, and customer communication

---

## üöÄ Tech Stack
- **Frontend:** React, Vite, Redux, React Router DOM, Tailwind CSS  
- **Backend:** Node.js, Express.js, MongoDB (Mongoose), Stripe API for payments  
- **Authentication:** JWT (JSON Web Token) for secure user sessions  
- **Real-time Chat:** Integrated customer-seller chat in the admin dashboard  
- **Others:** REST API, Cart management, Stripe payments, Order tracking  

---

## üéØ Key Features
- üõí Add/Remove products from cart  
- üë§ User registration & login (JWT Authentication)  
- üí≥ Secure payments with Stripe  
- üìä Admin dashboard for managing products, orders, and users  
- üí¨ **Real-time chat** between customers and sellers (in the dashboard)  
- üîç Product search and filtering  
- üì¶ Order management & stock control  

---

## üìÅ Project Structure
/frontend ‚Üí Customer-facing online shop
/backend ‚Üí Express API and business logic
/dashboard ‚Üí Admin panel with chat and management tools
